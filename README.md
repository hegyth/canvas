# Старт 

- npm run dev

# Управление:

- для удара нажмите лкм, удерживая отводите мышку в противоположную сторону от желаемого направления движения шара и отпускайте
- сила удара регулируется расстоянием, на которое отводится мышка при зажатой лкм в пределах canvas
- для смены цвета шара нажмите пкм и введите в поле желаемый цвет (в виде "red" или "#FF0000" или "rgb(255, 0, 0)")
